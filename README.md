# Emily’s Runaway Birthday Cupcake

A gentle browser game for young kids.

## Play
Open `index.html` in a browser.

## Controls
- Arrow keys / on-screen buttons for the maze and toilet-slide obstacle course
- Click/tap for decorating

## Levels
1. Catch the runaway cupcake ingredients in an easy maze.
2. Watch the silly giant cutscene.
3. Escape through the widened toilet-slide obstacle course.
4. Decorate the cupcake with Emily by freely placing ingredients wherever you want.

The game includes a cheerful background music loop. Browsers require the player to press Start before music can play.

No losing: bumps just bounce/reset gently.
